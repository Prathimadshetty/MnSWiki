M&S Securing Application checklist

Secure Application Checklist provides an easy to reference list of security controls that increases awareness and help engineering teams create more secure applications. It's a first step toward building a base of security knowledge around web application security. Consider using checklist to identify the minimum standard that is required to neutralize vulnerabilities in your critical applications.
